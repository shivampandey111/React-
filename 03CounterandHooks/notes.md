React controls the UI updation that is very difficult with JS like if there's a need to update multiple variables when one thing happens. 

This is done with the help of hooks. 

Like useState

When it changes, this looks for all of the variable named var_name and changes its states, the job is to sync UI and state


useState hook is used to manage the state of a component in functional components. 

Calling useState returns an array with two elements: the current state value and a function to update the state.
